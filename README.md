The Binance Alpha Box is now live!

This event uses the updated Alpha Box model, with the airdrop pool consisting of tokens from three projects: Hana Network（HANA）, Reservoir（DAM）, and Alaya Governance Token（AGT）. Users holding at least 242  Binance Alpha points can claim one token airdrop on the Alpha Events page. 

Upon claiming, users will be placed into different reward tiers and receive one of the following rewards: 700, 875, or 2500 HANA tokens; 700, 875, or 2500 DAM tokens; 2800, 3500, or 10000 AGT tokens. Rewards are distributed on a first-come, first-served basis.

If rewards are not fully distributed, the points threshold will automatically decrease by 5 points every 5 minutes.

Please note that claiming the airdrop will consume 15 Binance Alpha Points. Users must confirm their claim on the Alpha Events page within 24 hours; otherwise, it will be deemed that users have given up claiming the airdrop.
